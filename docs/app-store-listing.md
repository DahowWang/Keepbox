從 Line、Messenger、Safari 一鍵把整個網頁存進 Keepbox，離線也能完整開啟。用資料夾與標籤分類，內文搜尋秒找。

### 描述 Description
Keepbox 是你的網頁收藏盒——把任何 HTML 頁面完整存下來，離線隨時開啟、閱讀、整理。

不只是存連結，而是把「整頁畫面」存成檔案。網路斷了、原始網頁消失了，你收藏的內容依然在。

■ 一鍵存入
從 Line、Messenger、Safari 或任何 App 點「分享」，就能把網頁存進 Keepbox，整頁轉成可離線開啟的 HTML 檔。也可以直接貼上 HTML 原始碼或從「檔案」App 匯入。

■ 真實畫面預覽
每個檔案的縮圖都是網頁本身的縮影，不是死板的文件圖示。一眼就認出是哪一頁。

■ 三種瀏覽版型
網格、畫廊、清單即時切換，依你的習慣排列收藏。

■ 內文全文搜尋
不只比對檔名，連網頁內文都能搜，命中關鍵字即時高亮。

■ 資料夾與標籤
用資料夾與彩色標籤分類，工作、生活、旅遊、學習一目了然。

■ 沈浸式閱讀
點開即全螢幕渲染真實頁面，可調整字級、分享、在瀏覽器開啟。

■ 淺色／深色主題
跟隨系統或自選，白天夜晚都好讀。

■ 注重隱私
所有收藏都儲存在你的裝置上，不上傳、不追蹤、不需註冊帳號。

把值得留下的網頁，留在 Keepbox。

## English (en-US)

### Name (≤30 chars)
Keepbox: Save Webpages

### Subtitle (≤30 chars)
Save pages, read them offline

### Keywords (≤100 chars, comma-separated, no spaces)
HTML,webpage,save,offline,bookmark,archive,reader,clip,document,folder,tag,share,web,snippet

### Promotional Text (≤170 chars)
Save entire webpages from Line, Messenger, or Safari into Keepbox and open them offline anytime. Organize with folders and tags, and search inside the content.

### Description
Keepbox is your box for the web — save any HTML page in full and open, read, and organize it offline, anytime.

It doesn't just save links. It saves the whole page as a file. Even if the network drops or the original page disappears, what you kept is still there.

■ Save in one tap
Tap "Share" from Line, Messenger, Safari, or any app to save a page into Keepbox as an offline-ready HTML file. You can also paste raw HTML or import from the Files app.

■ Real-page thumbnails
Every file's thumbnail is a miniature of the actual page — not a generic document icon. Recognize what it is at a glance.

■ Three browsing layouts
Switch instantly between grid, gallery, and list to arrange your collection your way.

■ Full-text search
Search not only filenames but the page content too, with matches highlighted as you type.

■ Folders and tags
Organize with folders and colored tags — work, life, travel, study, all at a glance.

■ Immersive reading
Tap to open a full-screen render of the real page, adjust text size, share, or open in your browser.

■ Light & dark themes
Follow the system or choose your own — easy on the eyes day or night.

■ Privacy-first
Everything you save stays on your device. No uploads, no tracking, no account required.

Keep the webpages worth keeping — in Keepbox.

### What's New (first release)
Initial release:
• Import HTML by sharing from other apps
• Grid / gallery / list layouts
• Full-text content search
• Folders and tags
• Full-screen reading with adjustable text size
• Light / dark themes

## App Store Connect 其他必填欄位建議

- **分類 Category**: 主要 = 生產力工具 (Productivity)；次要 = 工具程式 (Utilities)
- **年齡分級**: 4+（無不當內容）
  - 注意：因內建 WebView 可開啟使用者匯入的任意 HTML，送審時「不受限制的網頁存取」問項建議據實回答；若僅渲染本機檔案、不提供任意瀏覽器，可評估是否勾選。
- **價格**: 免費（建議）
- **隱私政策網址**: 見 docs/privacy.html（透過 GitHub Pages 發佈）
- **支援網址**: 可暫用 GitHub repo 頁面 https://github.com/DahowWang/Keepbox
- **隱私營養標籤 (App Privacy)**: 選擇「不收集資料 (Data Not Collected)」——本 App 不蒐集任何使用者資料、無分析、無第三方 SDK。
